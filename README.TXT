Anleitung für Import nach Eclipse:

Projekt Verzeichnis anlegen, und darin ein Ordner in dem sich das GIT befindet.
(das ist nötig, da Eclipse auf der gleichen ebene wie das git noch einen Ordner anlegt)

Dann in Eclipse
 File->Import->general->existing Project
 
root Directory ist der Ordner vom Git

jetzt finish und das Projekt sollte laufen
